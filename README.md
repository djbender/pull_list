# Requirements

1. Postgres. Checkout [Postgres.app](http://postgresapp.com/) or
[homebrew](http://www.brew.sh): `brew info postgres`
1. ruby version 2.2.3 or greater

This is a Rails Edge project and as such many of the libraries are
prerelease.

2. `bin/rake db:setup`
2. `bin/rails server`
